# Project Styling and Role Guidelines

## Member 2 Role: Auth & User Service
* **Frontend**: Next.js, React, TypeScript, Tailwind
* **Backend**: Spring Boot 3, Spring Security, JWT (issuer), BCrypt
* **Database**: PostgreSQL (Spring Data JPA + Hibernate)
* **Extra Features**: Refresh token rotation

---

## Brand Color Palette: Cherish
Always use these color tokens when designing the UI:
* **Warm Ivory**: `#FFEFCD` (Primary background, light surfaces)
* **Natural**: `#A58E74` (Secondary text, borders, neutral elements)
* **Apricot**: `#E09132` (Accents, highlights, call-to-actions)
* **Fern**: `#424530` (Primary text, headers, dark buttons)
